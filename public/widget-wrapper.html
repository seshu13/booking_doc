How to Use This Widget

To embed this booking widget on your website, copy and paste the following code:

<!-- 1. Add this to your HTML where you want the button to appear --> <div id="booking-widget"></div> <!-- 2. Add these scripts at the end of your body tag --> <script> window.TREBOUND_DOMAIN = "https://bookingdoc-production.up.railway.app"; </script> <script src="https://bookingdoc-production.up.railway.app/embed.js"></script> <script> document.getElementById('booking-widget').appendChild( TreboundWidget.init({ buttonStyle: 'default', buttonText: 'Book Appointment', primaryColor: '#8B5C9E' }) ); </script>

Customization Options:

  • buttonStyle: 'default', 'outline', or 'minimal'
  • buttonText: Any text you want to display on the button
  • primaryColor: Any hex color code (e.g., '#8B5C9E')
